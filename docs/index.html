Feature list

Who What Where

 - See the IP addresses that your computer communicates with, sorted by amount of packets transmitted. Shows GeoIP and a ping result for each address. Can be used for determining a game server IP or monitoring suspicious internet connections.

Ping to Speech

 - Press a hotkey and listen to the result of a ping sent to a predefined IP address or hostname. Useful while playing online games.

Watchdog

 - Inspects network traffic and waits for a packet from a predefined list of IP addresses and source/destination ports and several protocols. Upon finding a match, a predefined message is played on the speakers. Can be used to as a notifier for an incoming connection, for example.

Trace and Visual Trace

 - Perform a trace-route for a given IP address or hostname. Once the trace is complete, a visual trace is available, which shows the route of your packet on a map.
